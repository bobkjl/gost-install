# gost-install
Gost-Utils的安装程序

### 使用方法
```shell
wget -N --no-check-certificate "https://raw.githubusercontent.com/bobkjl/gost-install/master/install.sh" && bash install.sh
```

### 功能说明
- 一键安装Gost
- 一键配置Gost服务端
- 一键配置Gost客户端
